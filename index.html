Apa itu Basis Data?

Basis data adalah kumpulan data yang terorganisir secara sistematis sehingga dapat dengan mudah diakses, dikelola, dan diperbarui.
Basis data memungkinkan penyimpanan informasi dalam jumlah besar dengan keamanan, keakuratan, dan kecepatan akses yang tinggi.

Apa itu Conceptual DB?

Conceptual Database adalah model basis data yang menggambarkan struktur informasi pada level konseptual atau abstrak.
Model ini menunjukkan hubungan antar data (entities, attributes, relationships) tanpa memperhatikan detail implementasi teknis.
Biasanya divisualisasikan dengan ERD (Entity Relationship Diagram).

Apa itu Physical Model?

Physical Model adalah representasi basis data yang sudah mengarah pada implementasi teknis di sistem manajemen basis data (DBMS).
Model ini memuat detail seperti tipe data, struktur tabel, indeks, primary key, foreign key, serta cara penyimpanan data secara fisik di media penyimpanan.

Penggunaan Basis data

  • Perbankan : Untuk informasi pelanggan, rekening, pinjaman, dan transaksi perbankan.
  • Penerbangan : Untuk pemesanan dan informasi jadwal. Maskapai termasuk di antara pertama yang menggunakan basis data secara geografis.
  • kartu kredit : Untuk pembelian dengan kartu kredit dan pembuatan laporan bulanan.
  • Penjualan : Untuk informasi pelanggan, produk, dan pembelian.
  • Online Store : Untuk data penjualan yang disebutkan di atas ditambah pelacakan pemesanan / pembuatan daftar rekomendasi on-line, dan pemeliharaan evaluasi produk online.
  • Universitas : Untuk data mahasiswa, dosen, mata kuliah, KRS, nilai akademik, perpustakaan, keuangan, dan administrasi kampus.
  • Pabrik/perusahaan : Untuk manajemen produksi, inventaris barang, data karyawan, jadwal kerja, rantai pasokan (supply chain), hingga analisis penjualan.
